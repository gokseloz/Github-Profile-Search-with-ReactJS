## Github Profile Search

- Search Github users by their name and view their information.

## Why did I work on this project?

- In order to improve my React skills

## Run it locally

1. Clone the repo
2. Install dependencies `npm install`
3. Run it `npm start`

## How did I do this?

- HTML, CSS, ReactJS
